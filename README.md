# App 项目管理程序

## 功能

* 项目版本管理
* 渠道管理
* 打包

## 部署

### 环境

* golang
* mongodb
* android sdk (Android打包)

### 依赖

* go get github.com/go-martini/martini
* go get github.com/martini-contrib/auth
* go get github.com/martini-contrib/binding
* go get github.com/martini-contrib/render
* go get gopkg.in/mgo.v2
